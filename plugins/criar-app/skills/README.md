# Skills do projeto

As skills criadas para este app moram aqui, cada uma em sua pasta com `SKILL.md` (+ `scripts/`, `references/`, `assets/` se precisar).

- Crie com **/skill <para que serve>** — que usa a skill **skill-creator** quando disponível.
- Regra de negócio que descreve um **fluxo repetível** (validação, cálculo, geração padronizada) é boa candidata a virar skill; ligue a skill às regras (RN-00x) que implementa.
- `description` da skill deve dizer **o que faz e quando usar**, de forma "puxada", para disparar bem.
